🧮 Java Calculator (GUI)
✨ Overview

This project is a desktop-based calculator application developed using Java Swing. It provides a simple yet effective interface for performing everyday arithmetic calculations.

Designed with clarity and usability in mind, this project is perfect for beginners learning Java GUI development and event handling.

🚀 Features

✔️ Perform basic arithmetic operations
✔️ Supports decimal calculations
✔️ Interactive button-based input
✔️ Lightweight and fast execution
✔️ Clean and minimal UI design

🛠️ Tech Stack
Technology	Description
Java	Core programming language
Swing	GUI framework
AWT	Event handling & UI components
